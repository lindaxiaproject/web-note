Command + F：代码查找 Command + C ：复制选中内容或整行 Command + N：新建文件 Command + O：打开文件 Command + W：关闭当前窗口 Command + B：显示/隐藏侧边栏 Command + Shift + P：代码缩进 Command + Shift + K ：删除当前行或选中的代码 Option + Shift + F : 格式化代码 Option + ⬆️ ：将当前行代码移动到上一行 Option + ⬇️ ：将当前行代码移动到下一行 Command + ⬅️ ：移动到当前行最前面 Command + ➡️ ：移动到当前行最末尾 Command + ⬆️ ：移动到文档第一行 Command + ⬇️ ：移动到文档最后一行 Option + Shift + ⬇️ ：向下复制当前行代码并且光标在新复制代码所在行 Command + / ：注释一行代码 Option + Shift + A ：注释一整段代码 Option + 移动鼠标指针 ：选择多行代码 Fn + ⬅️/➡️ ：光标定位到行首/行尾 Fn + F5 开始调试
